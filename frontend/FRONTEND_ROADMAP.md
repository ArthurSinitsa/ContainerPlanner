# План разработки фронтенда Container Planner

## 1. Стек технологий
- **UI & Логика:** React + TypeScript + Vite.
- **Роутинг:** React Router (SPA).
- **Стейт-менеджмент & API:** TanStack Query (кэширование, retry, состояния загрузки).
- **Визуализация 3D:** three.js через react-three-fiber.

## 2. Архитектура приложения
- `src/app/` — глобальные провайдеры (QueryClient, Toasts, Router).
- `src/features/` — бизнес-логика и фичи (расчеты, контейнеры).
- `src/components/` — переиспользуемые UI-компоненты (модалки, кнопки).
- `src/pages/` — сборка компонентов фичей в конечные страницы.
- `src/lib/` — утилиты, типы и API-клиент.
